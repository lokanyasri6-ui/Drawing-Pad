# 🎨 Drawing-Pad
A simple and interactive Drawing Pad built with Python and Tkinter. This application allows users to draw freely on a digital canvas using their mouse, making it a great beginner-friendly GUI project.

## 🚀 Features

- ✏️ Freehand drawing with mouse
- 🖥️ User-friendly graphical interface
- 🧹 Clear canvas functionality
- ⚡ Real-time drawing experience
- 🎨 Easy to customize with colors and brush sizes

## 🛠️ Technologies Used

- Python 3
- Tkinter
